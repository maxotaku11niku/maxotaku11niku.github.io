Magical Raceway Now On Github

Tada! My game Magical Raceway is now available to download as a Github release! Version 0.5 as well as every earlier version I archived is available for download. It's a retro style arcade racing game with magical girls I developed solo. Check out the page here if you want to learn more about how it is at the moment, and my plans for the game. I will use the rest of this post to talk about its history up until this point.

The early days: Before the Raceway became Magical

OutRun in Sega Arcade Gallery for the GBA.

Initially, I only wanted to replicate the sort of graphical effect used in classic arcade racing games like OutRun and Hang On because I thought it would be fun. Although, my actual point of reference was a Game Boy Advance port of some of Sega's classic arcade games made by the infamous THQ. Recreating this in Unity wouldn't be hard, right? I am targeting modern computers with gigahertz multicore processors and even integrated GPUs can render 2D graphics as if it were nothing. So all I have to work out is the maths, and it was some pretty complicated maths that I needed to figure out to get this to work.

To control the appearance of the road, five parameters need to be varied at runtime: two position coordinates, how much the road bends in both the horizontal and vertical directions and how much split there is between the two road segments. Well, the way to simulate the road width varying without stretching the road texture is to have two segments with an imaginary line marking the boundary between them, so that's what the last parameter controls: the difference in road offset between the two sides. If the road split value gets large enough, the road appears to split into a dual carriageway! Magical Raceway's road functions basically the same as this version of OutRun does in that respect.

The site is now live!

Hello! My Github Pages site is now active. This will be the place for most of the things I make. Free hosting is such a blessing, but it does come with some limitations, of course. Perhaps sometime in the future, this site may be hosted somewhere else, but for now, here it is.

This website was built with a text editor and a Python script to build the blog part. It gets the job done, hopefully without taking too long to load. If you're using a mobile, please do check if it displays correctly, because this website may be a bit biased towards desktop/laptop users. Sorry about that.
